hexo搭建记录

发表于2026-04-10|更新于2026-04-10
|阅读量:

部署环境

  • 安装 git
  • 安装 node
  • 安装 npm（安装 node 自动安装）
    安装 hexo 框架：% npm install hexo -g
  • 可能会出现权限问题，询问 qwen 新建全局文件夹、添加环境变量
  • 上述方法不太好，建议用 nvm 配置 。

文件介绍

  • 主题配置文件 _ config.landscape.yml

  • 框架配置文件 _ config.yml
    - 缩进很重要。不对会报错

  • themes 存放主题
  • source 源文件，存放编写的东西和文章

服务器

Hexo + GitHub Actions 双仓库自动部署到服务器 完整思路整理

Hexo + GitHub Actions 双仓库自动部署极简笔记

核心结构

  • 本地：ruolinglife/ 只写文章、改配置、改主题
  • 仓库 1（HexoBlog・私有）：存源码，触发构建
  • 仓库 2（HexoSite・公开）：存生成的网页，触发部署到服务器

完整流程

  1. 本地修改完成
  2. 双击 deploy.command 一键推送
  3. GitHub Action 自动构建网页 → 推送到仓库 2
  4. 仓库 2 Action 自动上传文件到你的服务器
  5. 网站完成更新
